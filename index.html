2. Lorsque vous êtes connectés en tant qu'administrateur plusieurs fonctionnalités sont disponibles comme :

  • Supprimer les demandes de postes
  • Modifier les statuts des demandes de postes
  • Accéder au Dashboard qui permet de gérer la base de données

Supprimer les demandes de postes

Aller sur la page d'accueil et cliquer sur le bouton rechercher :

Vous arriverez ensuite sur la page où sont listé les postes

Sur cette page, vous verrez tous les postes disponibles, si vous cliquez sur l'un d'entre eux, vous pourrez constater les demandes qui ont été formulé par des étudiants pour ce poste :

Afin de supprimer la demande de poste, vous devez cliquer sur l'icône de poubelle comme sur l'image ci-dessous :

Revenez sur la page ou vous pouvez voir les étudiants inscrits à un poste.

Afin du statut de la demande de poste, vous devez cliquer sur l'icône de crayon comme sur l'image ci-dessous :
